# CommercialFreeRadio

Command line executable (geschreven in C#) om reclame vrij webradio te luisteren via Sonos.

Command line usage:
```
Player options:
   /sonos[=<ip-address>]   Use sonos player. Default ip address=192.168.1.16
   /vlc                    Use VLC player 
Other options:
   [/verbose]              Print verbose
```
